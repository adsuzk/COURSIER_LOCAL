## 🔄 FLUX DE DONNÉES

### Commande (Exemple)
```
1. Client remplit formulaire
   📱 OrderScreen.kt
        ↓
2. Calcul distance/prix
   🌐 ApiService.kt → 🖥️ Serveur PHP
        ↓
3. Affichage résultat
   📱 OrderScreen.kt (prix)
        ↓
4. Client soumet
   📱 OrderScreen.kt
        ↓
5. Envoi au serveur
   🌐 ApiService.kt → 🖥️ Serveur PHP
        ↓
6. Confirmation
   ✅ Message succès
```

### Authentification
```
1. Client entre email/mdp
   📱 LoginScreen.kt
        ↓
2. Envoi au serveur
   🌐 ApiService.kt → 🖥️ Serveur PHP
        ↓
3. Réponse serveur
   🖥️ Session créée
        ↓
4. Stockage local
   💾 SessionManager.kt
        ↓
5. Navigation
   🧭 MainActivity → 🏠 HomeScreen
```
